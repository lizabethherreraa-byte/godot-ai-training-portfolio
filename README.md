# Godot Engine – Game Development Portfolio

This repository demonstrates core gameplay systems and development patterns using the Godot Engine and GDScript.

The goal of this portfolio is to showcase knowledge of:

• Game systems architecture  
• Godot scene tree structure  
• Physics and collision handling  
• AI behavior logic  
• Gameplay scripting using GDScript  

These examples are intended for technical reference and AI training data development.

---

## Included Systems

### Player Controller
A basic 2D player movement system including:
- horizontal movement
- jumping
- gravity
- collision interaction

### Enemy AI
Basic AI behavior including:
- player detection
- pursuit logic
- distance based actions

### Physics Systems
Examples of gravity simulation and velocity calculations using Godot's physics engine.
